# 🌐 Language Translation Tool

## 📌 Description
This project is developed as part of my Artificial Intelligence Internship at CodeAlpha.

It allows users to translate text between multiple languages using Google Translator API.

## 🛠️ Technologies Used
- Python
- Streamlit
- Deep Translator

## ⚙️ Features
- Translate text instantly
- Multiple language support
- Input validation (no numeric-only input)
- Clean and user-friendly UI

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the app:
   streamlit run app.py
